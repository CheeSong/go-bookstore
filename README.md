# Go Bookstore API

A simple REST API built in Go using GORM and Gorilla Mux.
Followed guide: https://www.youtube.com/watch?v=1E_YycpCsXw

## Purpose / Learning Goal

This project is a **personal learning project** to practice:

- Building REST APIs in Go  
- Using GORM for database management  
- Structuring a Go project with models, controllers, and routes  
- Handling environment variables securely with `.env`  

It is **not intended for production use**, but rather to improve of Go backend development.

Future Plans:
1. Complete with simple front-end.
2. Host the project on a server and enable access over the web.
